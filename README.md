An open-source Writing Assistant. Select text in any app and tap **Writing Assistant** in the context menu to instantly rewrite, proofread, summarize, and more — powered by AI running entirely on your device.

## ✨ Features

### ✍️ **Writing Tools**
- **Proofread** — fix grammar, spelling, and punctuation
- **Rewrite** — alternative phrasing with the same meaning
- **Friendly** — casual and warm tone
- **Professional** — formal and polished tone
- **Concise** — shorter and tighter, no filler
- **Summary** — 2–3 sentence summary
- **Key Points** — main ideas as bullet points
- **Table** — structured markdown table
- **List** — bullet or numbered list

### 🔗 **System Integration**
- **Works in every app** — appears natively in the text-selection toolbar
- **Replace in-place** — result is written back into the originating text field
- **Copy to clipboard** — one tap to copy the result

### 🤖 **On-Device AI**
- **Powered by LiteRT-LM** (Google's on-device inference runtime)
- **Qwen2.5 1.5B Instruct** — fast, capable, Apache 2.0
- **GPU-accelerated** with automatic CPU fallback
- **Multilingual** — responds in the same language as the input text
- **No cloud, no API keys, no subscriptions**

### 🎨 **Modern Design**
- **Material Design** with dynamic colors
- **Dark/Light theme**
- **Animated transitions** between tool states
